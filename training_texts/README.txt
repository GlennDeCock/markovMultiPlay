# training_texts/

Put any .txt files here to train the Markov chain.

The Markov chain generates ALL text in the game from these files:
- Scene text (the story paragraphs)
- Choice button labels
- Game over text
- Collision scene text

## Tips

- The more text, the richer the output.
- The chain doesn't need to make sense — poetic, fragmentary,
  absurdist, song lyrics, instruction manuals, old letters, anything works.
- Higher Markov order (set in the control panel) = output stays closer
  to the source. Order 1-2 = more chaotic. Order 3-4 = more coherent.
- You can add as many .txt files as you want. All are trained together.
- Click "Retrain from texts" in the control panel after adding new files.

## Included examples

- example_01.txt — absurdist poetic prose (moon, sparrows, Tuesdays)
- example_02.txt — object monologues (compass, kettle, hat, wall)

## What works well

- Repetitive structures ("X is a Y that Z")
- Lists and catalogues
- Things that are almost but not quite logical
- Old technical manuals with strange specific language
- Poetry without strict meter
- Transcribed conversation fragments
- Field notes for things that don't exist
