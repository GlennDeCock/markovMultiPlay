Node cards — printable sheets

Files added:

- node_card_template.html
- scripts/generate_node_cards.py
- node_cards_blank.html (generated by script)
- node_cards_prefilled.html (generated by script)
 - node_cards_blank_json.html (generated by script with `--format json`)
 - node_cards_prefilled_json.html (generated by script with `--format json`)

Quick instructions

- To print a blank set of cards (12 by default):

```bash
python scripts/generate_node_cards.py blank --count 12
```

This writes `node_cards_blank.html` to the project root. Open it in your browser and Print (Ctrl+P). Suggested settings: Portrait, Scale 100%, enable "Background graphics" if your browser hides the light lines.

- To generate one card page per node from your world JSON:

```bash
python scripts/generate_node_cards.py prefill data/world/example_world.json
```

This writes `node_cards_prefilled.html` with a page per node, prefilled with `id`, `title`, `description`, exits, tags and traces (where available).

JSON-style output

If you prefer to hand-write cards in a JSON-shaped form (so the text you write is already laid out like the JSON file), use the `--format json` flag.

Generate blank JSON-skeleton cards:

```bash
python scripts/generate_node_cards.py blank --count 12 --format json
```

Generate prefilled JSON blocks for each node:

```bash
python scripts/generate_node_cards.py prefill data/world/example_world.json --format json
```

These commands write `node_cards_blank_json.html` or `node_cards_prefilled_json.html` respectively.

Validator

To help avoid typos and disconnected nodes, a small validator script was added:

```bash
python scripts/validate_world.py nodes.json
```

This checks for missing `to` targets and lists nodes unreachable from `start_nodes`.
Add `--require-reciprocals` to flag one-way edges when you expect bidirectional links.

Printing tips

- Use A4 or Letter paper; Chrome/Edge previews the page breaks correctly.
- If you want multiple cards per sheet, you can reduce the count when generating blank cards and use your printer layout settings, or edit the HTML template to arrange multiple cards per page.

If you want I can also generate a PDF from the HTML for you, but that needs additional tooling. For now the HTML will print cleanly from any modern browser.
