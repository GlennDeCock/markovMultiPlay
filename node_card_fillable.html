Node Card — Fillable
1) id (lowercase_snake_case — required)
2) title (human-readable)
3) text (full description — write multiple lines)
4) exits — each row: label → to (node_id). Mark hidden (X)
labelto (node_id)hidden
5) tags (comma-separated short words)
6) notes (optional)
When finished: copy the JSON example (right) and replace values. Keep `id` unique. Use the validator: python scripts/validate_world.py nodes.json
How to transcribe into JSON
  1. Copy the values from fields into the JSON object keys shown below.
  2. For each exit row, add an object: { "label": "...", "to": "target_node_id", "hidden": true|false }.
  3. Keep strings quoted; escape internal quotes if needed.
  4. Paste the final object into the `nodes` array in `nodes.json`.
  5. Run `python scripts/validate_world.py nodes.json` to check targets and reachability.
JSON template (replace values)
{ "id": "your_node_id", "title": "Your Title", "text": "Full description text...", "exits": [ { "label": "enter market", "to": "market_hall", "hidden": false }, { "label": "secret ladder", "to": "roof_room", "hidden": true } ], "tags": ["tag1", "tag2"] }
Quick examples / tips
- Use short IDs: market_hall, plaza_fountain. - If an exit is two-way, add reciprocal exits on the other node's card. - Leave `drift` and `mutated` out — engine will manage these.
